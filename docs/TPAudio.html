Module TPAudio

Handles playing and loading sound effects and music.

Example usage:

local mySoundConfig = {
    file = "assets/Sounds/my-game.mp3",
}
local soundArrayConfig = {
    { file = "assets/Sounds/shoot-a.mp3", },
    { file = "assets/Sounds/shoot-b.mp3", },
    { file = "assets/Sounds/shoot-c.mp3", },
}

--DLC sound files must be in folder "Sounds/"
--this is used in function TPAudio.loadSound
local dlcSoundConfig = {
    fromDLC = "dlc-name",
    file = "my-game.ogg",
}

local dlcSoundArrayConfig = {
    {
        fromDLC = "dlc-name",
        file = "shoot-a.ogg",
    },
    {
        fromDLC = "dlc-name",
        file = "shoot-b.ogg",
    },
}

local gameSoundGroup = {
    [mySoundConfig] = true,
    [ soundArrayConfig[1] ] = true,
    [ soundArrayConfig[2] ] = true,
    [ soundArrayConfig[3] ] = true,
}

local TPAudio = require("tpcoronalibrary.TPAudio")

-- Preload sounds
TPAudio.loadSound(mySoundConfig)
TPAudio.loadSoundGroup(gameSoundGroup)

-- Play a sound, sounds are loaded if not preloaded
TPAudio.playSound(mySoundConfig, volume, pitch, onComplete)
TPAudio.playSound(soundArrayConfig)

-- Stoping sounds
TPAudio.stopSound(mySoundConfig)
TPAudio.stopWithFadeOut(mySoundConfig, 1000)
TPAudio.stopGroup(gameSoundGroup)
TPAudio.stopAll()

-- Stop and Dispose sounds
TPAudio.stop(mySoundConfig, true)
TPAudio.stopWithFadeOut(mySoundConfig, 1000, true)
TPAudio.stopAll(true)

-- Enable/Disable Audio
TPAudio.setEnableBgm(true)
TPAudio.setEnableSfx(false)

Tables

SoundConfig
Definition of a sound.

Fields:

  • channel int or table Number of the specified channel, or array of possible channels.
  • file string Filename of the sound, with extension.
  • isBackground bool Load sound using audio.loadStream if true.
  • loops int Number of times to loop | -1 (Loops indefinetly), -1 | 0 | 1 | ... | N
  • duration int Plays the sound for the specified miliseconds, then stops, regardless if finished or not.
  • fadein int Time in miliseconds to fade in.
  • fadeout int Time in miliseconds to fade out.
  • minimumInterval int Minimum time in miliseconds before playing this sound again. Optional.
  • dontStopSoundWhenDisablingSfx bool If set, this sound isn't stopped by stopSfx.
  • mediaAPIType string

    Optional flag to reduce audio lag on Android devices. Possible values:

    1. "sfx" files will be played with media.playSoundEffects. Cannot be stopped.

    2. "sound" files will be played with media.playSound. Can be stopped, paused, resumed but only one sound can be played at time.

SoundArray
Array of sounds.

Fields:

  • 1 SoundConfig The first sound in the array.
  • minimumInterval int Minimum time in miliseconds before playing this sound again. Optional.
  • mode string

    The pattern of usage of the array. Possible values:

    1. randomWithoutRepetition (default)
    2. sequentialLoop

Playing

playSound (soundInfo[, volume[, pitch[, onComplete]]])
Plays a sound.

Parameters:

  • soundInfo SoundConfig or SoundArray The sound to play. If it's an array, a config will be selected based on it's mode.
  • volume number 0.0 <= volume <= 1.0, with 1.0 being maximum volume. Defaults to 1. (optional)
  • pitch number 1 is normal. Double/half to increase/decrease by 12 semitones (one octave). Defaults to 1. (optional)
  • onComplete function Function to call when the audio finishes playing. Not reliable, and not called if sounds are disabled. (optional)

Returns:

  1. number The channel in which the sound was played.
  2. number The AL source in which the sound was played.

Or

    nil If no sound were played.
